# MyGreeting

WoW Classic Anniversary 전용 길드 관리 애드온입니다.  
길드원 자동 인사, 파티 인사, 버프/힐 감사, 장비점수 추적 기능을 제공합니다.

---

## 기능

### 길드 자동 인사
- 길드원 접속 시 시간대별 자동 인사 (아침/점심/저녁)
- 신규 길드원과 기존 길드원 구분 인사
- 재접속 감지 (5분 이내 재접 별도 처리)
- 레벨업 축하 메시지
- 던전 입장 감지 후 "무사히 돌고 득템하세요" 인사
- 길드챗에서 내 이름 언급 시 화면 상단 경보 표시

### 파티 자동 인사
- 파티/레이드 참가·퇴장 시 자동 인사

### 버프/힐 감사
- 버프·소환·부활 받았을 때 자동 감사 인사

### 장비점수 추적 (GearTracker)
- 길드원 타겟 또는 파티 결성 시 자동 인스펙트 → 장비점수 수집
- GearScore 애드온 캐시 우선 사용, 없으면 자체 공식 계산
- 특성(이중특성 포함) 자동 감지 및 표시
- 장비 목록 아이템 링크(로컬) / 아이템명(귓말·길드챗) 출력
- 수집 데이터 `SavedVariables`에 영구 저장

---

## 설치

1. 이 저장소를 다운로드합니다.
2. `MyGreeting` 폴더를 다음 경로에 복사합니다:
   ```
   World of Warcraft/_anniversary_/Interface/AddOns/MyGreeting/
   ```
3. 게임 실행 후 애드온 목록에서 **MyGreeting** 활성화

---

## 슬래시 명령어

| 명령어 | 설명 |
|---|---|
| `/mg` | 도움말 표시 |
| `/mg 장비` | 내 장비점수 확인 |
| `/mg 장비 [이름]` | 특정 길드원 장비점수 및 장비 목록 확인 |
| `/mg 장비순위` | 수집된 길드원 장비점수 순위 출력 |
| `/mg 장비초기화` | 장비 데이터 전체 초기화 |
| `/mg 장비디버그온` | 인스펙트 디버그 로그 활성화 |
| `/mg 장비디버그오프` | 인스펙트 디버그 로그 비활성화 |

---

## 길드 채팅 명령어 (길드원 누구나 사용 가능)

| 명령어 | 설명 |
|---|---|
| `!장비` | 내 장비점수 확인 (귓말 응답) |
| `!장비 [이름]` | 특정 길드원 장비점수 확인 (귓말 응답) |
| `!장비순위` | 길드원 장비점수 순위 (귓말 응답) |
| `!길드장비` | 길드원 장비점수 순위 (길드챗 출력) |
| `!길드장비 [이름]` | 특정 길드원 장비점수 (길드챗 출력) |

---

## 메시지 커스터마이즈

`/mg` 설정 패널에서 각 인사 메시지를 수정할 수 있습니다.  
`{name}`, `{level}`, `{zone}` 변수를 사용할 수 있습니다.

---

## 파일 구성

| 파일 | 역할 |
|---|---|
| `GuildGreeter.lua` | 길드 자동 인사, 언급 경보, 슬래시 명령어 |
| `GuildDB.lua` | 본캐/부캐 DB, 직업·종족 분포 |
| `PartyGreeter.lua` | 파티 자동 인사 |
| `BuffGreeter.lua` | 버프·소환·부활 감사 |
| `GearTracker.lua` | 장비점수 수집 및 조회 |
