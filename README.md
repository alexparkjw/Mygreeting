# MyGreeting

WoW Classic Anniversary 전용 길드 관리 애드온입니다.  
길드원 자동 인사, 파티 인사, 버프/힐 감사, 장비점수 추적 기능을 제공합니다.

A guild management addon for WoW Classic Anniversary.  
Features automatic guild greetings, party greetings, buff/heal thanks, and gear score tracking.

---

## 기능 / Features

### 길드 자동 인사 / Guild Auto Greetings
- 길드원 접속 시 시간대별 자동 인사 (아침/점심/저녁)
- 신규 길드원과 기존 길드원 구분 인사
- 재접속 감지 (5분 이내 재접 별도 처리)
- 레벨업 축하 메시지
- 던전 입장 감지 후 "무사히 돌고 득템하세요" 인사
- NovaWorldBuffs 연동: 오늘의 일퀘 던전/주간 전장 입장 시 특별 인사
- 길드챗에서 내 이름 언급 시 화면 상단 경보 표시

### 파티 자동 인사 / Party Auto Greetings
- 파티/레이드 참가·퇴장 시 자동 인사

### 버프/힐 감사 / Buff & Heal Thanks
- 버프·소환·부활 받았을 때 자동 감사 인사

### 장비점수 추적 / Gear Score Tracking
- 길드원 타겟 또는 파티 결성 시 자동 인스펙트 → 장비점수 수집
- GearScore 애드온 캐시 우선 사용, 없으면 자체 공식 계산
- 특성(이중특성 포함) 자동 감지 및 표시
- 직업별 장비점수 순위 필터 지원
- 수집 데이터 SavedVariables에 영구 저장

### 자리비움 / AFK
- 자리비움 상태에서 길드챗·귓말 응답 시 `<자리비움>` 자동 표시

## 설치 / Installation

1. `MyGreeting` 폴더를 다음 경로에 복사합니다.
   ```
   World of Warcraft/_anniversary_/Interface/AddOns/MyGreeting/
   ```
2. 게임 실행 후 애드온 목록에서 **MyGreeting** 활성화

## 인사 트리거 상세 / Greeting Triggers

### 길드 인사 (길드챗으로 전송)

| 상황 | 조건 | 메시지 |
|---|---|---|
| 신규 길드원 접속 | `knownMembers`에 없는 멤버 온라인 | `{name} 님 어서오세요 환영합니다` |
| 기존 멤버 접속 | 오프라인→온라인, 마지막 오프라인 5분 이상 경과 | `{name} 님 어서오세요` |
| 재접속 | 오프라인→온라인, 마지막 오프라인 5분 이내 | `{name} 님 재접하셨습니다 ~리하요` |
| 잠자리 멘트 | 길드챗에서 "자야/잘게/퇴근/들어가야..." 감지 | `{name} 님 푹쉬세요` |
| 레벨업 | 30초 주기 로스터 갱신 시 레벨 증가 감지 (최대 2레벨차) | `{name} 님 레벨업 축하해요 ({level})` |
| 아침 인사 | 매일 오전 6시 (당일 1회) | `좋은아침 입니다` |
| 점심 인사 | 매일 오전 11시 (당일 1회) | `식사들 하셨습니까? 맛점 하세요` |
| 던전 입장 | 존 변경 → 던전 지역 감지, 2시간 쿨타임 | `{name} 님 [{zone}] 무사히 돌고 득템하세요 :)` |
| 던전 입장 (일반 일퀘) | 위 + NWB 오늘의 일반 일퀘 던전과 일치 | `{name} 님 [{zone}] 오늘 일반 일퀘에요! 무사히 돌고 득템하세요 :)` |
| 던전 입장 (영웅 일퀘) | 위 + NWB 오늘의 영웅 일퀘 던전과 일치 | `{name} 님 [{zone}] 오늘 영웅 일퀘에요! 무사히 돌고 득템하세요 :)` |
| 전장 입장 (주간 전장) | 전장 지역 입장 + NWB 주간 전장 일치, 2시간 쿨타임 | `{name} 님 [{zone}] 주간 전장! 잘 싸우세요 :)` |

> 전장 인사는 NWB 주간 전장과 일치할 때만 발동합니다. 일반 전장 입장은 인사 없음.

### 파티 인사

| 상황 | 조건 | 메시지 |
|---|---|---|
| 파티 합류 (초대받은 경우) | 내가 초대된 파티에 합류 | `안녕하세요 반갑습니다` (파티챗) |
| 새 멤버 참가 | 파티에 새 멤버 입장 (내가 초대한 경우 제외) | `{name} 님 안녕하세요 반갑습니다` (파티챗) |
| 파티원 레벨업 | UNIT_LEVEL 이벤트, 최대 2레벨차 | `{name} 님 레벨업 축하합니다 ({level})` (파티챗) |
| 파티원 퇴장 | 함께한 시간 5분 이상일 때 | `담에 봬요 즐와세요` (귓말) |
| 파티 해산 / 내가 나갈 때 | 함께한 시간 5분 이상일 때 | `담에 봬요 즐와세요` (귓말, 남은 전원에게) |
| "수고" 감지 | 파티/공대챗에서 "수고" 포함 메시지, 60초 쿨타임 | `수고하셨습니다` (파티챗) |

|---|---|

| `/mg 현황` | 길드 접속 현황 |
| `/mg 레벨` | 레벨 분포 |
| `/mg 직업` | 직업 분포 |
| `/mg 종족` | 종족 분포 |

| `/mg 정보 [이름]` | 길드원 상세 정보 |

| `/mg 장비초기화` | 장비 데이터 전체 삭제 |
| `/mg 장비디버그온` | 인스펙트 디버그 로그 활성화 |
| `/mg 장비디버그오프` | 인스펙트 디버그 로그 비활성화 |

| `/mg remove [이름]` | 특정 멤버 신규 처리 |
| `/mg status` | 애드온 상태 |
| `/mg db` | 부캐 DB 출력 |

### 슬롯별 장비 조회
```
/mg 머리 [이름]     /mg 목 [이름]      /mg 어깨 [이름]
/mg 등 [이름]       /mg 가슴 [이름]    /mg 벨트 [이름]
/mg 다리 [이름]     /mg 발 [이름]      /mg 손목 [이름]
/mg 손 [이름]       /mg 손가락 [이름]  /mg 장신구 [이름]
/mg 주장비 [이름]   /mg 보조장비 [이름] /mg 원거리 [이름]
```
별칭도 동작합니다: `망토`=등, `주무기`=주장비, `보조무기`=보조장비, `신발`=발, `장갑`=손, `허리`=벨트

## 길드 채팅 명령어 / Guild Chat Commands

길드원 누구나 길드 채팅에서 `!명령어` 형식으로 사용합니다.  
응답은 **귓말**로 돌아옵니다. `!길드명령어` 형식으로 치면 **길드챗**에 출력됩니다.

### 현황 조회
| 명령어 | 설명 |
|---|---|
| `!현황` | 길드 접속 현황 |
| `!레벨` | 레벨 분포 |
| `!직업` | 직업 분포 |
| `!종족` | 종족 분포 |
| `!지역` | 지역별 현황 |
| `!인던` | 던전 입장 현황 |
| `!전문기술` | 전문기술 분포 |
| `!등급` | 등급별 목록 |
| `!정보 [이름]` | 길드원 상세 정보 |
| `!도움` | 길드챗 명령어 도움말 |

### 직업·종족·전문기술별 멤버 목록
| 명령어 | 설명 |
|---|---|
| `!전사` `!성기사` `!사냥꾼` `!도적` `!사제` | 직업별 접속중 멤버 |
| `!주술사` `!마법사` `!흑마법사` `!드루이드` `!죽기` | 직업별 접속중 멤버 |
| `!오크` `!언데드` `!타우렌` `!트롤` `!블엘` | 종족별 접속중 멤버 |
| `!인간` `!드워프` `!나엘` `!노움` `!드레나이` | 종족별 접속중 멤버 |
| `!대장` `!재봉` `!연금` `!기공` `!가세` `!보세` | 전문기술별 멤버 |
| `!약초` `!채광` `!마부` `!무두` `!요리` `!낚시` | 전문기술별 멤버 |

### 장비점수
| 명령어 | 설명 |
|---|---|
| `!장비` | 내 장비점수 (귓말 응답) |
| `!장비 [이름]` | 특정 길드원 장비점수 (귓말 응답) |
| `!장비길드` | 길드원 장비점수 순위 (귓말 응답) |
| `!장비전체` | 수집된 전원 장비점수 순위 (귓말 응답) |
| `!장비전사` | 전사 장비점수 순위 — 직업명 자유롭게 |
| `!길드장비` | 길드원 장비점수 순위 (길드챗 출력) |
| `!길드장비 [이름]` | 특정 길드원 장비점수 (길드챗 출력) |

### 슬롯별 장비
```
!머리   !목     !어깨   !등(망토)  !가슴
!벨트   !다리   !발     !손목      !손(장갑)
!손가락(반지x2)  !장신구(x2)
!주장비(주무기)  !보조장비(보조무기)  !원거리
```
`!머리 이름` 처럼 이름을 붙이면 해당 길드원 슬롯 조회.

사용 가능한 변수: `{name}`, `{level}`, `{zone}`

## 파일 구성 / File Structure

| 파일 | 역할 |
|---|---|
| `GuildGreeter.lua` | 길드 자동 인사, 언급 경보, 슬래시·길드챗 명령어 |
